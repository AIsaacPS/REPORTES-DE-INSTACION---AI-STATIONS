# REPORTES-DE-INSTACION---AI-STATIONS
Sistema de generación de reportes profesionales en LaTeX para documentar la instalación de estaciones sísmicas inteligentes. Incluye clase  LaTeX personalizada, templates parametrizados, scripts de visualización de datos sísmicos (MiniSEED → gráficos) y pruebas MQTT a AWS IoT  Core. Compilable con pdfLaTeX/Overleaf.
